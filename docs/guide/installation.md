# Installation

## Prérequis

- **Node.js** ≥ 23
- **npm** ≥ 11

## Cloner le dépôt

```bash
git clone https://github.com/thanatos-vf-2000/sap-devsec-scanner.git
cd sap-devsec-scanner
```

## Installer les dépendances

```bash
npm install
```

## Lancer l'application

```bash
# Production
npm start
# → http://localhost:3001
```
