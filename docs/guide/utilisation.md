# Utilisation

## Scanner un projet ZIP

1. Ouvrez l'application sur `http://localhost:3001`
2. Dans l'onglet **Scan**, choisissez **Upload ZIP**
3. Sélectionnez votre archive de projet SAP BTP
4. Cliquez sur **Lancer le scan**

![Vue de scan](/screenshots/scan.png)

## Scanner un répertoire local

Si l'application tourne sur le même serveur que votre projet :

1. Sélectionnez **Répertoire local**
2. Entrez le chemin absolu du répertoire
3. Lancez le scan

## Lire les résultats

Le rapport est organisé par catégorie de scanner avec un score de risque global.

![Rapport de scan](/screenshots/report.png)

## Historique

Les scans sont conservés en mémoire pour la session courante, accessibles via l'onglet **Historique**.

![Historique](/screenshots/history.png)
