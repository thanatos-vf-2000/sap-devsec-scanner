Installation ​

Prerequisites ​

  • Node.js ≥ 23
  • npm ≥ 11

Clone the repository ​

bash
git clone https://github.com/thanatos-vf-2000/sap-devsec-scanner.git
cd sap-devsec-scanner

Install backend ​

bash
npm install

Start the application ​

bash
# Production
npm start
# → http://localhost:3001
