À propos de SAP DevSec Scanner

SAP DevSec Scanner est un outil open-source de sécurité pour les développeurs SAP BTP. Il analyse vos projets SAPUI5/Fiori et CAP pour détecter les vulnérabilités de sécurité courantes.

Scanners disponibles
  • 🔵 UI5 Version Scanner - Détecte les versions obsolètes ou EOL
  • 🔵 NPM Security Scanner - Audit des dépendances npm (CVE)
  • 🔵 CAP Security Scanner - Contrôle d'accès, injection SQL, secrets
  • 🔵 UI5 Code Scanner - XSS, eval(), innerHTML, open redirect
  • 🔵 Secrets Scanner - Credentials, tokens JWT, clés API
